# Casa y Jardín - App móvil

Aplicación móvil en Flutter para las interfaces iniciales de Casa y Jardín.

## Pantallas incluidas

- Inicio de sesión.
- Página inicial del perfil administrador.

La app inicia siempre en la pantalla de inicio de sesión. Por ahora no hay backend ni validación real de usuarios; el botón de ingreso solo permite navegar a la interfaz del administrador para visualizar el flujo.

## Ejecutar en Codespaces

```bash
flutter pub get
flutter run
```

## Verificación sugerida

```bash
flutter analyze
flutter test
```
